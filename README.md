# 📚 Library Management System

A simple web-based Library Management System built using HTML, CSS, and JavaScript. This project allows users to manage books and borrowers efficiently with basic CRUD operations.

---

## 🚀 Features

- ➕ Add Book  
  Add new books with details like title, author, and ID.

- 📖 Issue Book  
  Assign books to borrowers and track issued records.

- ❌ Delete Book  
  Remove books from the system.

- 🔍 Search Book  
  Search books by title or author in real-time.

---

## 🛠️ Technologies Used

- HTML (Structure)
- CSS (Styling)
- JavaScript (Logic & Functionality)
- LocalStorage (Data Storage)
